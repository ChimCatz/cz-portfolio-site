Insights content folder structure

Use one folder per insight:

content/insights/
  insight-01/
  insight-02/
  insight-03/

Recommended files inside each insight folder:

- content.txt
- header-image.png
- image-01.png
- image-02.png
- image-03.png

You can change image extensions to .jpg, .jpeg, .webp, or .svg if needed.
Keep the numbering pattern for content images so future updates stay organized.
