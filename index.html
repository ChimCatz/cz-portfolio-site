Professional Overview

Building high-leverage data systems that turn manual CRM work into reliable, scalable workflows.

I'm a Data Analyst focused on automating workflows and improving data quality across CRM systems. I've built multiple Python-based tools that handle end-to-end data processing, from transformation and validation to search and reporting. My work includes developing an ETL pipeline that processes 10,000-50,000+ records per run, a fuzzy search engine with under 2 second query time, and a call attribution tool applying strict validation rules to ensure accurate conversion tracking. Across these projects, I've reduced manual processing time by 70-90% and cut data errors from around 10-15% down to under 3%.

Right now, I'm expanding into analytics and visualization, aiming to deliver not just clean data but actionable insights that improve decision-making speed and accuracy.

Fun Fact

Sudoku feels ancient, but it became popular worldwide only after Wayne Gould helped spread it in the early 2000s through newspapers like The Times. The name comes from Japanese and roughly means single number. A standard 9x9 Sudoku grid has about 6.67 x 10^21 possible completed solutions, so you will almost never see the same puzzle twice. Funny part: you do not need math skills at all, just logic and pattern spotting.

Minesweeper

Read the clues and manage the risk.

Trains probability thinking and quick decision-making under uncertainty.

Skills

Probability Thinking, Risk Management, Spatial Awareness, Decision Making, Attention to Detail

Fun Fact

Minesweeper is not just a casual game. In 2000, researchers proved that solving a Minesweeper board in general is NP-complete, meaning no known fast algorithm can solve all cases efficiently. It became famous worldwide because it shipped with early versions of Microsoft Windows, where it was meant to teach users mouse precision. Funny twist: some boards can require guessing, so even perfect logic does not guarantee a win.
