Who Wants to Be a Tech Master? - concept notes

Core gameplay loop:
answer -> advance -> fail -> reset

The player moves through a 30-question sequence one question at a time.
A correct answer advances the run.
A wrong answer resets the player back to the beginning.

Rank progression system:
- 1-5 -> Intern
- 6-10 -> Junior Developer
- 11-15 -> Software Engineer
- 16-20 -> Senior Engineer
- 21-25 -> Tech Lead
- 26-30 -> Chief Technology Officer

Difficulty scaling:
- Q1-Q10 -> Easy
- Q11-Q20 -> Medium
- Q21-Q30 -> Hard

The idea is inspired by quiz progression games where tension builds as the player climbs.

No scoring system yet.
Focus now is structure, progression, and overall experience.
Design direction is still open.
