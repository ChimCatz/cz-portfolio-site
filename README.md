# CZ Personal Website

Minimalist personal website built using Vite + React.

## Tech Stack

* React (Frontend)
* Vite (Build Tool)
* CSS (Custom Styling)

## Design System

* Colors:

  * #000000
  * #05081c
  * #0a0d36
  * #0f1042
  * #0e1d3e
* Font:

  * Proxima Nova (Primary)
  * Inter / Roboto (Fallback)

## Taglines

* "Improvise. Adapt. Overcome"
* "Data is LIFE"

## Goals (v1.0)

* Clean minimalist UI
* Fast load (<1s local)
* Modular components
* Scalable structure for backend integration

## Setup

```bash
npm install
npm run dev
```

## Roadmap

* v1.0 → Static portfolio
* v1.1 → Dynamic projects (JSON/API)
* v2.0 → Full backend integration
